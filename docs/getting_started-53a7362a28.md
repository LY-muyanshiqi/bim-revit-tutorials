# Revit水利工程建模入门指南

## 1. 软件安装与配置

### 1.1 获取Revit学生版
访问 Autodesk Education Community 官网，使用学生邮箱注册并下载 Revit 2025+ 版本。

### 1.2 系统要求
- 操作系统：Windows 10/11 64位
- 内存：16GB RAM（推荐32GB）
- 显卡：支持DirectX 11的独立显卡
- 存储：至少20GB可用空间

### 1.3 初始设置
1. 启动Revit，选择“新建项目”
2. 选择“建筑模板”或“结构模板”
3. 设置项目单位：米制（毫米）
4. 配置视图样板：创建水利工程专用视图

## 2. 界面导览

### 2.1 主要工作区
- **项目浏览器**：管理视图、图纸、族
- **属性面板**：编辑元素属性
- **视图控制栏**：控制显示模式、详细程度
- **快速访问工具栏**：常用工具快捷方式

### 2.2 水利工程专用工作空间
建议自定义工作空间：
1. 加载“结构”选项卡
2. 添加“分析”工具
3. 配置“水利”专用面板

## 3. 第一个水利构件：创建水闸底板

### 3.1 步骤说明
1. **选择工具**：转到“结构”选项卡 → “楼板”
2. **绘制边界**：使用矩形工具绘制 10m × 5m 底板轮廓
3. **设置属性**：
   - 类型：混凝土-300mm
   - 材质：C30混凝土
   - 厚度：800mm
4. **完成创建**：点击“完成编辑模式”

### 3.2 注意事项
- 确保底板高程设置为设计水位以下足够深度
- 考虑抗滑稳定，设置适当的底板厚度
- 添加排水孔和止水带构造

## 4. 下一步学习

完成基础入门后，建议：
1. 学习《基本操作指南》掌握更多建模技巧
2. 实践水闸完整项目建模
3. 探索参数化族创建方法
